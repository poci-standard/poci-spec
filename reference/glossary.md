# PoCI Glossary

**PoCI** – Proof of Correct Inference  
**Descriptor** – Objeto que describe un modelo de IA mediante commitments  
**Commit** – Registro inmutable de una inferencia  
**commitId** – Hash raíz del commit  
**Provider** – Genera inferencias y firma commits  
**Verifier** – Valida commits  
**Challenge** – Cuestiona un commit en PoCI v0
