# PoCI Specification

Repositorio oficial del estándar **PoCI (Proof of Correct Inference)** v0.

Este repositorio contiene:
- La especificación oficial PoCI v0 en PDF.
- La versión Markdown resumida.
- Historial de versiones.
- Glosario técnico.
- Vectores de test de conformidad.

## Estructura

```
/spec        → Especificación en Markdown
/docs        → PDF con la especificación completa
/versions    → Historial de versiones
/reference   → Glosario y notas
/tests       → Vectores de test PoCI v0
```

## Estado

Versión actual: **PoCI v0**

Esta versión define:
- Formato de commits
- Hashing canónico
- Firmas
- Reglas de serialización
- Invariantes del estándar
- Flujos básicos PoCI
